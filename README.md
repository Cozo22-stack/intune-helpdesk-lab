# Microsoft Intune Help Desk Lab

A 5-scenario hands-on help desk lab built from scratch on a live Microsoft 365 Intune tenant. No guided walkthroughs — every scenario was troubleshot and documented independently.

## Scenarios

| Ticket | Scenario | Category |
|--------|----------|----------|
| LAB-001 | Device flagged non-compliant — BitLocker not enabled | Endpoint Compliance |
| LAB-002 | User reports app not installed — 7-Zip Win32 deployment | App Deployment |
| LAB-003 | User locked out — BitLocker recovery key retrieval | Account Reset |
| LAB-004 | Lost device — remote retire and MDM unenrollment | Device Management |
| LAB-005 | User blocked from sign-in — Conditional Access policy | Conditional Access |

## Each ticket document includes
- Ticket summary and environment details
- Step-by-step actions taken
- Screenshots at every stage
- Resolution notes and lessons learned

## Tools Used
- Microsoft Intune
- Microsoft Entra ID
- Windows 11 Enterprise (Oracle VirtualBox)
- PowerShell
- Microsoft Win32 Content
